Spendsight Setup

Welcome to Spendsight

Let's connect your budget tracker to Google Sheets. This quick setup will have you tracking expenses in minutes.

You'll need:

1 A Google account
2 A Google Cloud project with Sheets API enabled
3 A Google Sheet to store your transactions

Enable Google Sheets API

First, we need to enable the Google Sheets API in your Google Cloud Console.

1 Click the button below to open Google Cloud Console
2 Create a new project or select an existing one
3 Click "Enable" to activate the Sheets API
Open Google Cloud Console

Download Credentials

Now create and download your API credentials file.

1 Go to Credentials in the Cloud Console
2 Click Create Credentials → Service Account
3 Name it anything (e.g., "spendsight") and click Create
4 Click on the service account → Keys → Add Key → JSON
Create Service Account
